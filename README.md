# snapp_food

<h2>how it's run:</h2>

<p>1. clone project</p>

<p>2. update packages</p>

```
composer update
```

<p>3. copy .env file</p>

```
cp .env.example .env
```

<p>4. Generate key</p>

```
php artisan key:generate
```

<p>5. Generate Jwtkey</p>

```
php artisan jwt:secret
```

<p>6. Instal npm packages</p>

```
npm install
```

<p>7. npm run dev</p>

```
npm run dev
```

<p>8.create migrate & seed</p>

```
php artisan migrate --seed
```

<p>9. Run Server</p>

```
php artisan serve
```
